# Fly for Peace – Official Website

The official web platform for the **Fly for Peace – World Peace Movement**, a global initiative dedicated to promoting unity, harmony, and peaceful coexistence among nations.

---

## Overview

Fly for Peace is a global movement that transforms the sky into a symbol of unity, delivering the message:

"Air is not for war; Air is to breathe."

This website serves as the central digital platform for communicating the vision, leadership, and global activities of the movement while enabling public engagement and financial support.

---

## Features

* Responsive and accessible web design
* Structured content for global initiative presentation
* Leadership and organizational information
* Donation system integration with secure payment processing
* Contact form with real-time email delivery
* News and updates section for ongoing activities
* Scalable backend architecture

---

## Technology Stack

**Frontend**

* HTML5
* CSS3
* JavaScript

**Backend and Services**

* Firebase (Database and optional hosting)
* Resend API (Email service integration)

---

## Firebase Integration

Firebase is utilized for:

* Storing contact form submissions
* Managing dynamic or user-generated data
* Supporting scalable backend infrastructure

---

## Email Service Integration

The application integrates the Resend API to:

* Process contact form submissions
* Send automated email notifications
* Ensure reliable and efficient email delivery

---

## Payment Integration

The platform includes integration with the PayHere payment gateway to facilitate secure online donations, including support for custom contribution amounts.

---

## Project Structure

* `/src` – Core application source files
* `/assets` – Static resources including images and branding
* `/components` – Reusable UI components
* `/services` – External service integrations (Firebase, Resend)
* `/pages` – Application views and sections
